Your building tools

Nodes!

Join nodes to create the flows and see your story paths, texts, choices, all in your sight.

Customized paths

Adapt to user choices with events, stats and requirements. Give them some stat and use it to show or hide choices, or to point to a node.
For example, when user clicks on the choice "Take the sword", you can register an event to add a sword to the user. Later, in a node where the sword could be useful, show the answer "Use the sword" only if the user has it.

Text interpolations

Add user info in the middle of the story.
For example, if your story has points of some type (like an exam), you could show this points in a node using hashtags: "You have #amountOfPoints points!"
